Za pravljanje ove igrice koriscena je
lokalna sql baza u koju su se skladistili podaci.

Pokretanje ove igrice na nekom drugom kompjuteru
nije moguce osim ukoliko ne postoji identicna baza.

Da bi se napravila sql baza potrebno je skinuti
SQL Server Management Studio(express) i napraviti
svoju bazu.

U tu bazu treba dodati sva polja koja sadrzi i
lokalna baza.

I u kodu je potrebno promeniti naziv baze iz koje
se uzimaju podaci.
